# How to install

---

## Create Python Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
.venv\Scripts\activate
```

### macOS

```bash
source venv/bin/activate
```

### Deactivate Environment Command

```bash
deactivate
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Streamlit

```bash
streamlit run app.py
```
